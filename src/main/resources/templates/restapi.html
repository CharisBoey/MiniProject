Rest API Usage

This page is intended for developers who desire to write applications that interact with InkIt.'s API

Individual User Data:

To extract an individual user's data, replace {email} will the email of the user. If link is used, you will be redirected to a page that ends with "%7Bemail%7D". Replace this value with the email of the user.

https://vttp-miniproject-inkit.up.railway.app/Home/InkItAPI/{email}
All User Data:

To extract all user's data, use the link shown below. Click to display Json data of all users.

https://vttp-miniproject-inkit.up.railway.app/Home/InkItAPI?q=*
